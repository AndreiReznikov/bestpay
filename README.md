# Технологии

- Vite — сборка и дев-сервер
- Web Components — нативные кастомные элементы
- TypeScript — типизация

# Работа с проектом

- "install": Установка зависимостей
```
npm install
```

- "dev": Режим разработки
```
npm run dev
```
